Eclipse Rich Client Platform is a framework for developing desktop application software. The Eclipse Integrated Development Environment is itself written with this framework. The forthcoming entries are about development of a 'feature' for the RCP program OpenChrom which is used for visualization of mass spectrometry data.

Feature Structure

The Feature under development, 'alignment' includes five plugins, which in eclipse are represented as project folders:

  • net.openchrom.chromatogram.xxd.process.supplier.alignment
  • net.openchrom.chromatotram.xxd.process.supplier.alignment.cbi
  • net.openchrom.chromatogram.xxd.process.supplier.alignment.feature
  • net.openchrom.chromatogram.xxd.process.supplier.alignment.fragment.test
  • net.openchrom.chromagotram.xxd.process.supplier.alignment.ui

Core Functionality

The actual functionality that the feature shall provide can be found in the first plugin 'alignment'. In it's 'src' folder, the packages 'alignment', 'alignment.core', 'alignment.model', 'alignment.preferences' and 'alignment.settings' are contained. 'core' and 'model' contain the actual functonality.

UI Functionality

The other plugin of interest is 'alignment.ui' which contains in it's source folder a number of packages that together host the code for the graphical user interface. The entry point for the GUI part of the plugin in question can be found in the 'alignment.ui' plugin and more specific in the 'alignment.ui.editors' package.
