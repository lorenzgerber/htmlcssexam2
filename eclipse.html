Eclipse Rich Client Platform is a framework for developing desktop application software. The Eclipse Integrated Development Environment is itself written with this framework. The forthcoming entries are about development of a plugin for the RCP program OpenChrom which is used for visualization of mass spectrometry data.

Plugin Structure

The plugin under development, 'alignment' includes five project folders:
  • net.openchrom.chromatogram.xxd.process.supplier.alignment
  • net.openchrom.chromatotram.xxd.process.supplier.alignment.cbi
  • net.openchrom.chromatogram.xxd.process.supplier.alignment.feature
  • net.openchrom.chromatogram.xxd.process.supplier.alignment.fragment.test
  • net.openchrom.chromagotram.xxd.process.supplier.alignment.ui

The actual functionality that the plugin shall provide can be found in the first project folder, 'alignment'. In it's src folder, the packages 'alignment', 'alignment.core', 'alignment.model', 'alignment.preferences' and 'alignment.settings' are contained. 'core' and 'model' contain the actual functonality.

The other project folder of interest ist 'alignment.ui' which contains in it's source folder a number of packages that together host the code for the graphical user interface. The entry-point, code-wise (altough mostly boiler plate) is the 'Activator' class in 'alignment.ui' package.
